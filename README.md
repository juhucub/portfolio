# Jacob Gardner Portfolio

A production-ready one-page portfolio built with React, TypeScript, Vite, Tailwind CSS v4, and Framer Motion.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Structure

- `src/components` reusable UI and animation primitives
- `src/sections` page sections
- `src/data` typed content sources
- `src/types` shared interfaces
- `public/projects` placeholder project covers
